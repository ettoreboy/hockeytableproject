hockeytableproject
==================
Code for an awesome Hockey board table made with Arduino Uno.

## Features
- Detects 3 goals
- Play goalie music

The Hardware
------
The table is 80x70 cm and was made with plastic glass, old Subbuteo doors and laserzz.
